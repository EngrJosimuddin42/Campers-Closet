# Campers Closet ⛺

**Campers Closet** is a feature-rich mobile application designed for outdoor enthusiasts and campers. It helps users organize their camping gear, manage packing lists, and keep track of upcoming travel events so they are always "All Packed" and ready to go.

## ✨ Key Features
* **Smart Dashboard:** Real-time summary of active camping lists and upcoming events.
* **Gear Management:** A digital closet to manage and categorize your camping equipment.
* **Event Planning:** Track upcoming trips and travel schedules.
* **Templates:** Ready-to-use packing templates for Summer Camps, Tours, and more.
* **Interactive UI:** Smooth carousels for active lists and a modern, clean design.

## 🛠️ Tech Stack
* **Framework:** [Flutter](https://flutter.dev) (v3.41.4)
* **Language:** [Dart](https://dart.dev)
* **State Management:** [GetX](https://pub.dev/packages/get) (Reactive state & Dependency Injection)
* **Local Storage:** [GetStorage](https://pub.dev/packages/get_storage)
* **UI Enhancements:** * `flutter_screenutil` (Responsive design)
    * `carousel_slider` (Interactive banners)
    * `google_fonts` (Custom typography)
    * `flutter_svg` (Vector asset support)

## 📸 UI Overview
- **Good evening greeting:** Personalized user experience.
- **Active Lists:** Auto-playing carousel showing current progress.
- **Shop Now Banners:** Integration for gear upgrades.
